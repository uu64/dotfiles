## Prerequisites

This repository runs on macOS or ubuntu and depends on the following.

- make
- Python 3.x
- python3-apt (Linux)
- aptitude (Linux)

## Setup

- Install dependencies.
  ```sh
  $ pip install -r requirements.txt
  ```
